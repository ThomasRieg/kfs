inspired from kfs project-line at 42-school.

currently implemented: vga output, serial output/input for terminal, memory paging, lazy memory allocation, interrupts and hardware exceptions handling, processes, memory isolation between processes, pipes, syscalls, basic tty job control, basic signals (no restorer or signal queue yet), basic filesystem (no vfs or permission checks yet), various syscalls necessary to make glibc work.

you can open kfs_diagram on the excalidraw website to get a visual description of the project.

for 42 students: this is equivalent to kfs-7 (minus some features), I didn't find a kfs so advanced on github yet so we have to be exemplary, you can take inspiration, look at our diagram, that we will try to keep up to date, to understand what the subjects want. If you need help understanding a concept you can hit me up on slack.
Also I have to be very clear, this is a big project, very few students chose to do it and it gives a LOT of xp, so use the ctrl+c and ctrl+v shortcuts VERY responsibly, the reputation of our school is at stake.
